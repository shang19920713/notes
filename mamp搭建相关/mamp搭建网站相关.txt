nginx报404错误 需要在nginx.conf配置文件里加入如下代码

if (!-e $request_filename) {
    rewrite  ^(.*)$  /index.php?s=/$1  
    last;
}

如果有路由前缀存在 可在配置文件中加入这段代码 其中dms就是前缀 到时候需要替换成实际前缀

location ~ .*\.(gif|jpg|jpeg|png|bmp|swf|flv|ico|woff|svg)$ {
                rewrite ^/dms/(.*)$ /$1 last;
                expires 30d;
        }

        location ~ .*\.(js|css)?$ {
                rewrite ^/dms/(.*)$ /$1 last;
                expires 1d;
        }
