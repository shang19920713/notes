#TP框架linux访问项目报404错误：很有可能是vhost下的.conf文件下 location /里缺少下面这段内容
`location / { 
   if (!-e $request_filename) {
           rewrite  ^(.*)$  /index.php?s=/$1  last;
   }
}`
